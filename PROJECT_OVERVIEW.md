# Project Overview

## 项目定位

HYG 是一个面向材料研究场景的智能工作台，用于把材料知识检索、结构文件管理、结构计算与结果分析串起来。

## 主要实现内容

### `chat.py`

- 提供网页聊天前端
- 识别用户意图
- 路由到本地检索、CIF 下载、CIF 解析、计算、结果分析等不同链路

### `matesim_dft.py`

- 读取候选结构
- 调用 MatterSim 做结构弛豫
- 输出每原子能量、最大受力、收敛状态等指标
- 对多个候选做优先级快排

### `parse_cif.py`

- 解析 CIF 内容
- 输出 JSON 结构
- 对比两个 CIF 之间的关键结构变化

### `download_mp_cif.py`

- 从 Materials Project 按元素体系或化学式检索结构
- 下载前 N 个 CIF 到本地

## 计算逻辑

当前实现的不是最终物性预测，而是前期预筛逻辑。

### 计算流程

1. 检索候选结构
2. 生成或读取 CIF
3. 对结构进行弛豫
4. 提取数值指标
5. 根据指标进行候选排序

### 当前使用的关键指标

- 每原子能量
- 最大受力
- 收敛情况
- 晶格参数

### 排序逻辑的意义

当前排序是“继续研究优先级排序”，不是最终性能结论。  
它主要用于帮助用户把后续计算资源优先投入到更稳定、更合理的候选上。

## 适用场景

- 本地材料检索
- 候选材料预筛
- CIF 下载与解析
- 演示型结构计算工作流

## 边界

当前版本不直接等同于：

- 完整 DFT / QE 工作流
- 完整超导 Tc 预测
- 最终实验结论系统
