# Demo Script

## 演示顺序

### 1. 本地检索

提示词：

```text
请检索 TA15，并展示它的材料信息。
```

讲解词：

这里先演示本地材料库检索能力。系统会先命中本地结构化 JSON，而不是直接去外部数据库，说明它可以基于企业自有材料数据工作。

### 2. 线上 CIF 下载

提示词：

```text
下载 Nb-Ti 前五个 CIF 到本地。
```

讲解词：

这里演示从外部结构库检索并下载 CIF。说明系统不仅能回答问题，还能真正把结构文件拉到本地供后续复用。

### 3. 开放讨论

提示词：

```text
我想找更值得继续研究的层状材料候选，应该优先关注哪些特性？
```

讲解词：

这里演示系统可以先分析再计算，不会把所有问题都误判成任务执行。

### 4. 数值计算

提示词：

```text
Nb–Ti–Sn 系
- Nb–Ge 系
- Mo–Re 系
对这三类进行快排。
```

讲解词：

这里演示多体系候选快排。系统会先检索候选结构，再进行结构弛豫，输出关键数值指标并做优先级排序。

### 5. 结果分析

提示词：

```text
为什么当前第一名更值得继续验证？当前排序依据是什么？
```

讲解词：

这里演示系统可以围绕已有计算结果继续解释，不只是给一个排序表，而是能解释为什么这样排。
